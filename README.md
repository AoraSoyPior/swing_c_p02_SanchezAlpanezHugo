# Gestión de Apartamentos Turísticos AlmeriaCruising

## Descripción del proyecto
[cite_start]Este proyecto consiste en una aplicación de escritorio desarrollada para la gestión de inmuebles turísticos[cite: 1, 2]. [cite_start]La herramienta permite el registro de arrendadores y la configuración detallada de viviendas, incluyendo validaciones de seguridad para datos sensibles y un sistema de cálculo de precios dinámico basado en las características del inmueble[cite: 3].

## Contenido de la publicación
[cite_start]Siguiendo los requisitos de la práctica, el repositorio se estructura en los siguientes directorios:
* [cite_start]**desarrollo**: Carpeta que contiene el proyecto completo con el código fuente de la aplicación.
* [cite_start]**ejecutable**: Directorio donde se ubica el archivo JAR para la ejecución directa del software.
* [cite_start]**documentación técnica**: Espacio destinado al Javadoc generado para las clases del proyecto.
* [cite_start]**documentos**: Carpeta que incluye el análisis de usabilidad de la interfaz.

## Desarrollo del proyecto
[cite_start]Durante el desarrollo se han implementado las siguientes características técnicas[cite: 2, 3]:
* [cite_start]Interfaz principal con barra de menú, mnemónicos y aceleradores para una navegación eficiente[cite: 2].
* [cite_start]Ventana de alta mediante JDialog modal con validación de DNI y teléfono mediante expresiones regulares[cite: 3].
* [cite_start]Componentes dinámicos que muestran u ocultan elementos según la interacción del usuario[cite: 3].
* [cite_start]Sistema de resumen de datos mediante JTabbedPane antes de la confirmación del registro[cite: 3].
* [cite_start]Implementación de Look and Feel con soporte para temas claro y oscuro mediante la librería FlatLaf[cite: 2].

## Despliegue
[cite_start]Para ejecutar la aplicación desde el archivo ejecutable, siga estas instrucciones:

### Windows
1. [cite_start]Localice el archivo .jar en la carpeta ejecutable.
2. [cite_start]Realice doble clic sobre el archivo para iniciar la aplicación (requiere Java 17 o superior).

### Línea de comandos
1. [cite_start]Abra una terminal o consola del sistema.
2. [cite_start]Ejecute el comando: java -jar AlmeriaCruising.jar.
